Codificador / Decodificador Base64

Pega cualquier texto para convertirlo a o desde el formato Base64 al instante.

URL-Safe cambia `+` y `/` por `-` y `_` y puede eliminar el relleno `=`. Codificar como URL aplica `encodeURIComponent` a la cadena Base64.
