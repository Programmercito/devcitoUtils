Navegador no compatible

En móvil, abre la página en Safari (iOS 15+) o Chrome para Android
Asegúrate de que la URL comienza con https://

¿Cómo funciona el generador de llaves SSH?

Entropía real del mouse

Cada movimiento del cursor genera datos impredecibles usados como semilla criptográfica, maximizando la aleatoriedad de la llave generada.

Criptografía RSA en el navegador

Usa la Web Crypto API nativa del navegador para generar llaves RSA de 2048, 4096 u 8192 bits. Nunca se envía nada a un servidor.

Formato estándar OpenSSH

Descarga la llave privada (id_rsa) y la pública (id_rsa.pub) en formato compatible con OpenSSH y authorized_keys.
