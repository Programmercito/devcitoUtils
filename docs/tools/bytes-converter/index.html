Conversor de Texto y Bytes

Convierte texto a su representación en bytes y bytes a texto en diferentes formatos.

Conversión de Texto a Array de Bytes

En la computación de bajo nivel y el desarrollo de protocolos, los datos a menudo no se manejan como texto plano, sino como secuencias de números. Un **Array de Bytes** es la representación fundamental de la información en memoria. Nuestra herramienta te permite visualizar exactamente cómo se traduce tu texto a nivel binario, decimal o hexadecimal.

Formatos de Salida

Obtén la representación en **Decimal** (0-255), **Hexadecimal** (00-FF) o **Binario** (00000000-11111111). También puedes elegir el separador (coma, espacio, prefijos 0x) para copiar los datos directamente en tu código fuente.

Codificación de Caracteres

Soportamos **UTF-8**, que es el estándar universal. Es vital entender que un mismo carácter (especialmente emojis o letras con tildes) puede ocupar múltiples bytes, y esta herramienta te permite inspeccionar ese comportamiento.

¿Cuándo es útil esta herramienta?

  • Debugging de Red: Analiza paquetes de datos crudos capturados en el tráfico de red.
  • Programación de Microcontroladores: Define constantes de texto para pantallas LCD o almacenamiento en EEPROM.
  • Criptografía: Prepara datos para ser procesados por algoritmos que requieren entradas en forma de bytes.
  • Aprendizaje: Entiende cómo se codifican los diferentes alfabetos y caracteres especiales a nivel de máquina.

Dato técnico: En UTF-8, los caracteres ASCII estándar ocupan 1 byte, mientras que los caracteres internacionales y emojis pueden ocupar de 2 a 4 bytes. Esta herramienta te ayuda a visualizar esa diferencia de forma clara.
